# Device.prototype.isTablet

The read-only **`isTablet`** property of the `Device` interface is
either a boolean stating if the device is a tablet (like an iPad), or `null` if the this is not known.


## Value

Either a boolean stating if the device is a tablet (like an iPad), or `null` if the this is not known.
