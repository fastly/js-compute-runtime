# console.info()

The **`console.info()`** method outputs an informational message to the console.

## Syntax

```js
info(obj1)
info(obj1, /* …, */ objN)
```

### Parameters

- `obj1` … `objN`
  - : A list of JavaScript objects to output. The string representations of each of these
    objects are appended together in the order listed and output.

### Return value

None `undefined`.
