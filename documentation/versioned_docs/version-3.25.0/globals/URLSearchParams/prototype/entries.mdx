# URLSearchParams.entries()

The **`entries()`** method of the
`URLSearchParams` interface returns an
iterator allowing iteration through all key/value
pairs contained in this object. The iterator returns key/value pairs in the same order as they appear in the query string. The key and value of each pair are
string objects.

## Syntax

```js
entries()
```

### Parameters

None.

### Return value

Returns an iterator.
