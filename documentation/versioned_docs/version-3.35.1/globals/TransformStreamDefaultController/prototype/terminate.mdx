# TransformStreamDefaultController.terminate()

The **`terminate()`** method of the `TransformStreamDefaultController` interface closes the readable side and errors the writable side of the stream.

## Syntax

```js
terminate()
```

### Parameters

None.

### Return value

None (`undefined`).
