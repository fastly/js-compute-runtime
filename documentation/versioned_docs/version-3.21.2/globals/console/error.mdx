# console.error()

The **`console.error()`** method outputs an error message console.

## Syntax

```js
error(obj1)
error(obj1, /* …, */ objN)
```

### Parameters

- `obj1` … `objN`
  - : A list of JavaScript objects to output. The string representations of each of
    these objects are appended together in the order listed and output.

### Return value

None `undefined`.
