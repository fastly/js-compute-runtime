# Device.prototype.isTouchscreen

The read-only **`isTouchscreen`** property of the `Device` interface is
either a boolean stating if the device's screen is touch sensitive, or `null` if the this is not known.


## Value

Either a boolean stating if the device's screen is touch sensitive, or `null` if the this is not known.
