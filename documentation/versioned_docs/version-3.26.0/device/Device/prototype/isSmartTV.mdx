# Device.prototype.isSmartTV

The read-only **`isSmartTV`** property of the `Device` interface is
either a boolean stating if the device is a smart TV, or `null` if the this is not known.


## Value

Either a boolean stating if the device is a smart TV, or `null` if the this is not known.
