# Response.statusText

The **`statusText`** read-only property of the [`Response`](../../../globals/Response/Response.mdx) interface contains the status message corresponding to the HTTP status code in [`status`](../../../globals/Response/prototype/status.mdx).

For example, this would be `OK` for a status code `200`, `Continue` for `100`, `Not Found` for `404`.

## Value

A [`String`](../../../globals/String/String.mdx) containing the HTTP status message associated with the response.
The default value is "".
