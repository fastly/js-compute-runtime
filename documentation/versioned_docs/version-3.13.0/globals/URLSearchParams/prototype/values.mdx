# URLSearchParams.values()

The **`values()`** method of the `URLSearchParams`
interface returns an iterator allowing iteration
through all values contained in this object. The values are string
objects.

## Syntax

```js
values()
```

### Parameters

None.

### Return value

Returns an iterator.
