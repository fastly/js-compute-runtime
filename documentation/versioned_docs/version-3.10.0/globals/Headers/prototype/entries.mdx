# Headers.entries()

The **`Headers.entries()`** method returns an iterator allowing to go through all key/value pairs
contained in this object. The both the key and value of each pairs are `String` objects.

## Syntax

```js
entries()
```

### Parameters

None.

### Return value

Returns an iterator.
