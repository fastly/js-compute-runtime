# ReadableStreamDefaultController.enqueue()

The **`enqueue()`** method of the
`ReadableStreamDefaultController` interface enqueues a given chunk in the
associated stream.

## Syntax

```js
enqueue(chunk)
```

### Parameters

- `chunk`
  - : The chunk to enqueue.

### Return value

None (`undefined`).

### Exceptions

- `TypeError`
  - : Thrown if the source object is not a `ReadableStreamDefaultController`.
