# Set.prototype.delete()

The **`delete()`** method removes a specified value from a
`Set` object, if it is in the set.

## Syntax

```js
delete(value)
```

### Parameters

- `value`
  - : The value to remove from `Set`.

### Return value

Returns `true` if `value` was already in
`Set`; otherwise `false`.
