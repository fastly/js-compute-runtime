# TextDecoder.ignoreBOM

The **`ignoreBOM`** read-only property of the `TextDecoder` interface is a `Boolean` indicating whether the [byte order mark](https://www.w3.org/International/questions/qa-byte-order-mark) is ignored.

## Value

`true` if the byte order mark is ignored; `false` otherwise.
