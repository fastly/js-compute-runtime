# BigInt.prototype.valueOf()

The **`valueOf()`** method returns the wrapped primitive value
of a `BigInt` object.

## Syntax

```js
bigIntObj.valueOf()
```

### Return value

A BigInt representing the primitive value of the specified `BigInt` object.
