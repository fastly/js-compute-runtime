# WeakSet.prototype.add

The **`add()`** method appends a new object to the end of a
`WeakSet` object.

## Syntax

```js
add(value)
```

### Parameters

- value
  - : Required. The object to add to the `WeakSet` collection.

### Return value

The `WeakSet` object.
