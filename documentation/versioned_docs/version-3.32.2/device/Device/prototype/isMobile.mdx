# Device.prototype.isMobile

The read-only **`isMobile`** property of the `Device` interface is
either a boolean stating if the device is a mobile phone, or `null` if the this is not known.


## Value

Either a boolean stating if the device is a mobile phone, or `null` if the this is not known.
