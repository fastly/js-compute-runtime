# Number.isFinite()

The **`Number.isFinite()`** method determines whether the passed value is a finite number â€” that is, it checks that a given value is a number, and the number is neither positive [`Infinity`](../../globals/Infinity.md), negative `Infinity`, nor [`NaN`](../../globals/NaN.md).

## Syntax

```js
Number.isFinite(value)
```

### Parameters

- `value`
  - : The value to be tested for finiteness.

### Return value

The boolean value `true` if the given value is a finite number. Otherwise `false`.
