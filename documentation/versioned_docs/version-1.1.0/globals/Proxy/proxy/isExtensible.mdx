# handler.isExtensible()

The **`handler.isExtensible()`** method is a trap for
[`Object.isExtensible()`](../../../globals/Object/isExtensible.md).

## Syntax

```js
new Proxy(target, {
  isExtensible(target) {
  }
});
```

### Parameters

The following parameter is passed to the `isExtensible()` method.
`this` is bound to the handler.

- `target`
  - : The target object.

### Return value

The `isExtensible()` method must return a boolean value.

## Description

The **`handler.isExtensible()`** method is a trap for
[`Object.isExtensible()`](../../../globals/Object/isExtensible.md).

### Interceptions

This trap can intercept these operations:

- [`Object.isExtensible()`](../../../globals/Object/isExtensible.md)
- [`Reflect.isExtensible()`](../../../globals/Reflect/isExtensible.md)

Or any other operation that invokes the `[[IsExtensible]]` [internal method](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Proxy#object_internal_methods).

### Invariants

If the following invariants are violated, the trap throws a [`TypeError`](../../../globals/TypeError/TypeError.md) when invoked.

- `Object.isExtensible(proxy)` must return the same value as
  `Object.isExtensible(target)`.
