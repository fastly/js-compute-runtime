# WorkerLocation

The **`WorkerLocation`** interface defines the absolute location of the script executed by the Worker. Such an object is initialized for each worker and is available via the [`location`](../../globals/location.md) property obtained by calling `globalThis.location`.

## Instance properties

- [`WorkerLocation.href`](../../globals/WorkerLocation/href.md) _**read-only**_
  - : Returns a string containing the serialized [`URL`](../../globals/URL/URL.md) for the worker's location.
- [`WorkerLocation.protocol`](../../globals/WorkerLocation/protocol.md) _**read-only**_
  - : Returns the [`protocol`](../../globals/URL/prototype/protocol.md) part of the worker's location.
- [`WorkerLocation.host`](../../globals/WorkerLocation/host.md) _**read-only**_
  - : Returns the [`host`](../../globals/URL/prototype/host.md) part of the worker's location.
- [`WorkerLocation.hostname`](../../globals/WorkerLocation/hostname.md) _**read-only**_
  - : Returns the [`hostname`](../../globals/URL/prototype/hostname.md) part of the worker's location.
- [`WorkerLocation.origin`](../../globals/WorkerLocation/origin.md) _**read-only**_
  - : Returns the worker's [`origin`](../../globals/URL/prototype/origin.md).
- [`WorkerLocation.port`](../../globals/WorkerLocation/port.md) _**read-only**_
  - : Returns the [`port`](../../globals/URL/prototype/port.md) part of the worker's location.
- [`WorkerLocation.pathname`](../../globals/WorkerLocation/pathname.md) _**read-only**_
  - : Returns the [`pathname`](../../globals/URL/prototype/pathname.md) part of the worker's location.
- [`WorkerLocation.search`](../../globals/WorkerLocation/search.md) _**read-only**_
  - : Returns the [`search`](../../globals/URL/prototype/search.md) part of the worker's location.
- [`WorkerLocation.hash`](../../globals/WorkerLocation/hash.md) _**read-only**_
  - : Returns the [`hash`](../../globals/URL/prototype/hash.md) part of the worker's location.

## Instance methods

- [`WorkerLocation.toString`](../../globals/WorkerLocation/toString.md)
  - : Returns a string containing the serialized [`URL`](../../globals/URL/URL.md) for the worker's location. It is a synonym for [`WorkerLocation.href`](../../globals/WorkerLocation/href.md).
