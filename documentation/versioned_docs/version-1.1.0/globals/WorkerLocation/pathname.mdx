# WorkerLocation.pathname

The **`pathname`** property of a [`WorkerLocation`](../../globals/WorkerLocation/WorkerLocation.md) object returns the [`pathname`](../../globals/URL/prototype/pathname.md) part of the worker's location.

## Value

A string.
