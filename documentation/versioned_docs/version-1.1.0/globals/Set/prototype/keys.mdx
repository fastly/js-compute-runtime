# Set.prototype.keys()

The **`keys()`** method is an alias for the [`values()`](../../../globals/Set/prototype/values.md) method.

## Syntax

```js
keys()
```

### Return value

A new iterator object containing the values for each element in the given
`Set`, in insertion order.
