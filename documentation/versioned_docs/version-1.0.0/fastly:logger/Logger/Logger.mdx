# `Logger()`

The **`Logger` constructor** lets you connect your Compute application to a [Fastly Named Logger](https://developer.fastly.com/learning/integrations/logging/).

**Note**: Can only be used when processing requests, not during build-time initialization.

## Syntax

```js
new Logger(name)
```

> **Note:** `Logger()` can only be constructed with `new`. Attempting to call it without `new` throws a [`TypeError`](../../globals/TypeError/TypeError.mdx).

### Parameters

- `name` _: string_
  - The Fastly Logger which should be associated with this Logger instance
  
### Return value

A new `Logger` object.
