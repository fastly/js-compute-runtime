# URLSearchParams.keys()

The **`keys()`** method of the `URLSearchParams`
interface returns an iterator allowing iteration
through all keys contained in this object. The keys are string
objects.

## Syntax

```js
keys()
```

### Parameters

None.

### Return value

Returns an iterator.
