# Response.text()

The **`text()`** method of the [`Response`](../../../globals/Response/Response.mdx) interface takes a [`Response`](../../../globals/Response/Response.mdx) stream and reads it to completion.
It returns a promise that resolves with a [`String`](../../../globals/String/String.mdx).
The response is _always_ decoded using UTF-8.

## Syntax

```js
text()
```

### Parameters

None.

### Return value

A Promise that resolves with a [`String`](../../../globals/String/String.mdx).
