# Request.text()

The **`text()`** method of the `Request` interface
reads the request body and returns it as a promise that resolves with a `String`.
The response is _always_ decoded using UTF-8.

## Syntax

```js
text()
```

### Parameters

None.

### Return value

A Promise that resolves with a `String`.
