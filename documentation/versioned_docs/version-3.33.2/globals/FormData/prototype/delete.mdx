# FormData.delete()

The **`delete()`** method of the `FormData` interface removes all key/value pairs with the given name from the `FormData` object.

## Syntax

```js
delete(name)
```

### Parameters

- `name`
  - : The name of the key you want to delete.

### Return value

`undefined`
