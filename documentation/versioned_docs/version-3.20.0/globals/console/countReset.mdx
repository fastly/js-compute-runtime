# console.countReset()

The **`console.countReset()`** method resets counter used with `console.count()`.

## Syntax

```js
countReset()
countReset(label)
```

### Parameters

- `label` __optional__
  - : A string. If supplied, `countReset()` resets the count for
    that label to 0. If omitted, `countReset()` resets the default counter to
    0.

### Return value

`undefined`.
