# Device.prototype.isGameConsole

The read-only **`isGameConsole`** property of the `Device` interface is
either a boolean stating if the device is a video game console, or `null` if the this is not known.


## Value

Either a boolean stating if the device is a video game console, or `null` if the this is not known.
