# Response.blob()

The **`blob()`** method of the [`Response`](../Response.mdx) interface takes a [`Response`](../Response.mdx) stream and reads it to completion. It returns a promise that resolves with a `Blob`.

## Syntax

```js
blob()
```

### Parameters

None.

### Return value

A `Promise` that resolves with a [`Blob`](../../../globals/Blob/Blob.mdx).

### Exceptions

- `TypeError`
  - : Thrown if the response body has already been consumed.
