# URLSearchParams.has()

The **`has()`** method of the `URLSearchParams`
interface returns a boolean value that indicates whether a parameter with the
specified name exists.

## Syntax

```js
has(name)
```

### Parameters

- `name`
  - : The name of the parameter to find.

### Return value

A boolean value.
