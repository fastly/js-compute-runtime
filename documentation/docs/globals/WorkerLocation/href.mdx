# WorkerLocation.href

The **`href`** property of a [`WorkerLocation`](../../globals/WorkerLocation/WorkerLocation.md) object returns a string containing the serialized [`URL`](../../globals/URL/URL.md) for the worker's location.

## Value

A string.
