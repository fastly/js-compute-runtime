# WorkerLocation.hostname

The **`hostname`** property of a [`WorkerLocation`](../../globals/WorkerLocation/WorkerLocation.md) object returns the [`hostname`](../../globals/URL/prototype/hostname.md) part of the worker's location.

## Value

A string.
