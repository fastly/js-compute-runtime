# Symbol.toStringTag

The **`Symbol.toStringTag`** well-known symbol is a string valued property that is used in the creation of the default string description of an object. It is accessed internally by the [`Object.prototype.toString()`](../../globals/Object/prototype/toString.md) method.

## Value

The well-known symbol `Symbol.toStringTag`.
