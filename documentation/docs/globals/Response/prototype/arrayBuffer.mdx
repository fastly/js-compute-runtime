# Response.arrayBuffer()

The **`arrayBuffer()`** method of the [`Response`](../../../globals/Response/Response.md) interface
takes a [`Response`](../../../globals/Response/Response.md) stream and reads it to completion. It returns a promise
that resolves with an [`ArrayBuffer`](../../../globals/ArrayBuffer/ArrayBuffer.md).

## Syntax

```js
arrayBuffer()
```

### Parameters

None.

### Return value

A promise that resolves with an [`ArrayBuffer`](../../../globals/ArrayBuffer/ArrayBuffer.md).
