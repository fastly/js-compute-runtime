# Response.body

The **`body`** read-only property of the [`Response`](../../../globals/Response/Response.md) interface is a [`ReadableStream`](../../../globals/ReadableStream/ReadableStream.md) of the body contents.

## Value

A [`ReadableStream`](../../../globals/ReadableStream/ReadableStream.md), or else `null` for any `Response` object [constructed](../../../globals/Response/Response.md) with a null `body` property, or for any actual HTTP response that has no body.
