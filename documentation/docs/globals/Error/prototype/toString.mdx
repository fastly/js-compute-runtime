# Error.prototype.toString()

The **`toString()`** method returns a string representing the
specified [`Error`](../Error.md) object.

## Syntax

```js
toString()
```

### Return value

A string representing the specified [`Error`](../Error.md) object.

## Description

The [`Error`](../Error.md) object overrides the [`Object.prototype.toString()`](../../Object/prototype/toString.md)
method inherited by all objects.
