# Error.prototype.name

The **`name`** data property of `Error.prototype` is shared by all [`Error`](../Error.md) instances. It represents the name for the type of error. For `Error.prototype.name`, the initial value is `"Error"`. Subclasses like [`TypeError`](../../../globals/TypeError/TypeError.md) and [`SyntaxError`](../../SyntaxError/SyntaxError.md) provide their own `name` properties.

## Value

A string. For `Error.prototype.name`, the initial value is `"Error"`.

## Description

By default, [`Error`](../Error.md) instances are given the name "Error". The `name` property, in addition to the [`message`](./message.md) property, is used by the [`Error.prototype.toString()`](./toString.md) method to create a string representation of the error.
